类型转换发生于需要将不同类型的源值转换为目标值的过程

## 各种值都具有类型
值有以下几种
* 字面量
* 常量
* 变量

### 字面量有其默认类型
* 一个字符串字面量的默认类型是预声明的string类型。
* 一个布尔字面量的默认类型是预声明的bool类型。
* 一个整数型字面量的默认类型是预声明的int类型。
* 一个rune字面量的默认类型是预声明的rune（亦即int32）类型。
* 一个浮点数字面量的默认类型是预声明的float64类型。
* 如果一个字面量含有虚部字面量，则此字面量的默认类型是预声明的complex128
* 其它类型的字面量要么是组合类型要么是定义类型，其类型就是其本身

### 常量
* 类型不确定常量类型是源值的类型
* 声明的类型就是其类型

### 变量
声明的类型就是其类型

## 类型转换发生的地方
* 赋值过程
* 表达式计算过程
当表达式计算的两项中至少有一项为常量时，
```go
func main() {
	var a int
	var b uint8 = 255
	a = int(1 + b)
	fmt.Printf("%v", a) // a = 0
}
```
* 函数和方法出入参
